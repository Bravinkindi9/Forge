# Future Ideas (Postponed, Not Rejected)

Ideas below are intentionally out of scope for Version 1. They are recorded here instead of implemented, to protect V1's focus. Nothing here should be built without an explicit decision to revisit it.

- Analytics (usage stats, post performance tracking)
- Scheduling posts for later publishing
- Direct LinkedIn API integration (auto-posting)
- Notifications / reminders
- Multi-user support, authentication, OAuth
- Team/collaboration features
- Search across past entries
- Categories / tags for entries
- AI memory across sessions (remembering the user's voice/style over time)
- Semantic search over past entries or drafts
- Analytics dashboards
- Gamification (streaks, badges)
- A UI for editing prompts (prompts stay developer-maintained in `lib/prompts.ts`)
